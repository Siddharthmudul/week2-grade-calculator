# Week 2 - Grade Calculator

## Objective

Create a Python program that calculates a student's total marks, percentage, and grade.

## Files

- grade_calculator.py - Main Python program
- test_students.txt - Sample input
- results_sample.txt - Sample output
- README.md - Project documentation
- .gitignore - Ignore unnecessary files

## Requirements

- Python 3.x

## How to Run

```bash
python grade_calculator.py
```

## Grade Criteria

| Percentage | Grade |
|------------|-------|
| 90-100 | A |
| 80-89 | B |
| 70-79 | C |
| 60-69 | D |
| Below 60 | F |
